Frequently Asked Questions

About my 3 week course +

My 3-week web development course at PITP "People Information Technology Program", is Initialized and being Taught by "Mehran University of Engineering & Technology" by Best Mentors like Sir Chettan Kumar and Other faculty. In these 3 weeks We Focused on:

  • HTML5 and semantic markup
  • CSS3 with Flexbox and Grid layouts
  • JavaScript fundamentals and DOM manipulation
  • Form validation and user interaction
  • Responsive web design principles
What I learned in 3 weeks +

During the intensive 3-week course, I gained practical skills in:

  • HTML5: Semantic HTML structure, Basic Tags, Headings, Sections, Forms, Ordered & Un-Ordered Lists, Select tag and much more tags
  • CSS3: Advanced styling, animations, responsive design with media queries
  • JavaScript: Variables, functions, events, DOM manipulation, form validation
  • Project Development: Made Hands-On projects like Personal Portfolio
  • Problem Solving: Debugging code and finding solutions to development challenges

This website is a demonstration of the skills acquired during the course, showcasing form validation, interactive elements, and responsive design.

What I will Learn Next? +

After completing the foundational course, We plan to continue learning:

  1. Advanced JavaScript (ES6+ features, async programming)
  2. Front-end frameworks like React or Vue.js
  3. Back-end development with Node.js
  4. Database management and API development
  5. Version control with Git and GitHub

My goal is to become a full-stack developer capable of building complete web applications.
